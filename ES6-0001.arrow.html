ARROW FUNCTION

También conocidas como funciones lambda, arrow function, o flechas, son abreviaciones de funciones utilizando el operador =>.

Es una forma de hacer funciones más compactas en JS ya que permite tener funciones en una sola línea.

FUNCIÓN NORMAL

var x = function(a){

return a;

}

console.log(x('Hola'));

FUNCIÓN FLECHA CON UN PARÁMETRO

var y = a => a;

console.log(y('hola arrow'));

FUNCIÓN FLECHA CON VARIOS PARÁMETROS

var y = (a, b) => a+' '+b;

console.log(y('hola', 'varios'));

FUNCIÓN FLECHA SIN PARÁMETROS

var y = () => 'Arrow sin parámetros';

console.log(y());

LLAMAR A LA FUNCIÓN DESDE OTRA FUNCIÓN

setTimeout(() => alert('SetTimeOut'), 1000);

FUNCIÓN FLECHA CON VARIOS PARÁMETROS

var arrowZ = () => {

a = 'hola';

b = 'arrow Z';

alert(a+' '+ b);

}

arrowZ();
